# The PR Butler — Ship-Ready Automation

Create a **SKILL.md** that automates an entire pre-commit workflow — turning messy code into a ship-ready pull request in one pass. Use **any IDE** with AI agent support (VS Code, Cursor, IntelliJ, Windsurf, etc.).

---

## Quick Start

```bash
cd scaffold/website
npm install
npm run build      # Must pass before you proceed
npm run dev        # Open the URL to see the Task Manager app
```

Requires **Node 18+**. Once the site builds and runs, you're ready.

---

## The Challenge

We provide a **deliberately flawed** scaffold website. Your job is to write a SKILL.md that instructs an AI agent to fix everything and prepare a clean PR.

**Read [`tech_challenge.md`](tech_challenge.md) for the full brief** — what's broken, what your Skill must do, how to structure it, and how you'll be graded.

---

## Submitting

1. **Fork** this repository
2. Create your SKILL.md (see `tech_challenge.md` Section 3 for placement)
3. Run your Skill against the scaffold to verify it works
4. **Commit frequently** — we review your progression
5. Submit your **fork URL** via the AI Hub Portal
6. In the notes field, **paste the full contents of your SKILL.md** — ensures evaluation even if repo access is restricted
